# MY-CALCULATOR
A simple yet stylish calculator built using HTML, CSS, and JavaScript. This project demonstrates core front‑end development skills, combining clean design with functional interactivity. - Perform basic arithmetic operations: addition, subtraction, multiplication, division.
- Responsive layout for different screen sizes
- Interactive buttons with hover and click effects
- Clear and reset functionality for smooth user experience
- Lightweight and easy to run in any modern browser
Tech Stack
- HTML → Structure of the calculator
- CSS → Styling, layout, and responsive design
- JavaScript → Logic for calculations and dynamic interactions
