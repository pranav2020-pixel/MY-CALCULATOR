# My Calculator

A simple web-based calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and is hosted using GitHub Pages.
---
 ## Live Demo
 
 https://pranav2020-pixel.github.io/MY-CALCULATOR/
---

## Features

Basic arithmetic operations:
Addition (+)
Subtraction (−)
Multiplication (×)
Division (÷)
Modulus (%)
Clear all (AC) and delete (DEL) functionality
Decimal support
Simple and clean user interface

---

## Technologies Used

HTML – structure of the calculator

CSS – styling and layout

JavaScript – calculator logic and interactivity

## How to Run Locally

Clone the repository:

git clone https://github.com/pranav2020-pixel/MY-CALCULATOR.git

Open the folder:

cd MY-CALCULATOR

Open index.html in your browser.
